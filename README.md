# 💱 Sistema de Câmbio

## 📌 Descrição
Aplicação desenvolvida em Java para simulação de conversão de moedas, utilizando conceitos de Programação Orientada a Objetos (POO).  
O sistema permite calcular valores convertidos entre diferentes moedas com base em taxas pré-definidas.

## 🚀 Funcionalidades
- Conversão entre moedas (ex: EUR, USD, BRL)
- Cálculo automático com base em taxa de câmbio
- Interface via console
- Estrutura modular e orientada a objetos

## 🛠️ Tecnologias utilizadas
- Java
- IntelliJ IDEA

## 🧠 Conceitos aplicados
- Programação Orientada a Objetos (OOP)
- Encapsulamento
- Organização em classes
- Separação de responsabilidades
